# Aanshi-cdac-.
Regional language Translation:-This system uses a pretrained model NLLB-200 model to translate Bhojpuri and Maithili into Hindi or English,then another pretrained model mBART is used for summarization. The pipeline works in Two stages first the Regional language was converted into hindi/english then the translated text was used for summarization. Deployed using a Gradio interface, allowing users to translate and summarize regional language text in a single click.
