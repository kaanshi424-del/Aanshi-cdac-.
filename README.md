# Aanshi-cdac-.
Regional language Translation:-This system uses a pretrained model NLLB-200 then fine tuning of the model was done on the created dataset to translate Bhojpuri and Maithili into Hindi or English,then another pretrained model mBART  is used and fine tuned on the created dataset for summarization. The pipeline works in Two stages first the Regional language was converted into hindi/english then the translated text was used for summarization. Deployed using a Gradio interface, allowing users to translate and summarize regional language text in a single click.
